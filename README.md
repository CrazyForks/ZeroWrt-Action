## 功能特性
### 基本功能
| 基本 | 状态 | 基本 | 状态 |
|------|------|------|------|
| kmod 内核模块安装 | ✅ | 全锥型 NAT（NFT、BCM 双方案） | ✅ |
| AES 硬件加速 | ✅ | 构建优化（O3） | ✅ |
| GPU 硬件加速 | ✅ | 在线 OTA 升级（squashfs） | ✅ |
| HDMI 终端输出 | ✅ | 固件重置（squashfs） | ✅ |
| BBRv3 拥塞控制 | ✅ | CPU 超频支持 | ✅ |
| KVM 虚拟化支持 | ✅ | Shortcut-FE（支持 UDP 入站） | ✅ |
| LLVM-BPF 支持 | ✅ | LRNG（v57） | ✅ |
| NGINX & CURL HTTP3/QUIC 支持 | ✅ | NAT6 | ✅ |
